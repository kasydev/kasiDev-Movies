# kasiDev Movies
*A simple app for displaying list of movies.*

  ## Tools and Technologies used:
- React.js.
- React-Icons.
- TailwindCSS.
- TypeScript.
- Redux ToolKit
- The Movie database API
- Vercel 
Live Website => https://kasi-dev-movies.vercel.app/
